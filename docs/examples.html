Recursive Fibonacci

A classic recursive algorithm that showcases Juno's explicit type casting system. Notice how we cast between int and uint â€” Juno requires explicit conversions, preventing subtle bugs from implicit type coercion. 

import Io;

uint fib(int n) {
  if (n < 2) {
    return uint<n>;
  }
  return fib(n-1) + fib(n-2);
}

void main() {
  int n = 6;
  Io.println("fib("^^string<n>^^") = "^^string<fib(n)>);
}

Key features demonstrated:

  • Explicit type casting: uint<n> and string<n> show Juno's strict type system
  • String concatenation: The ^^ operator joins strings together
  • Recursion: Clean, readable recursive function definition
  • Module imports: Simple import Io; gives access to I/O functions

Output: 

fib(6) = 8
