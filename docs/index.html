Juno

A modern programming language for the JVM

Installation

Get started with Juno by running the installer: 

bash <(curl -fsSL https://raw.githubusercontent.com/2LoPalma/juno-lang/master/install.sh)

Hello World

Your first Juno program: 

import Io.{println};

void main() {
  Io.println("Hello, Juno!");
}

Run it with: juno run hello.juno

Features

• Compiles to JVM bytecode
• Modern syntax
• Easy to learn
• Lightweight and fast
