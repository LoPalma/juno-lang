Juno

Old look, new feel

Juno brings a familiar, C-like syntax but adds a modern twist. With variant types, pattern matching, and powerful module systems, you can write expressive, type-safe code without losing the simplicity that makes C iconic. 


import Io;

void main() {
  string|int x = 0; // integer
  x = "string"; // string
}

Slimware oriented

In Juno, everything is modular. Instead of using heavy class hierarchies, you work with lightweight modules for data and behavior. This design makes Juno ideal for slim, fast, and minimal applications — perfect for embedded or high-performance environments. 


import String;
import Io;

void main() {
  string hi = "hi!";
  int len = String.length(hi);
}

Modern tools

Juno’s compiler is built for clarity. No cryptic build commands — just juno run file. You can generate .j files for the JVM or run programs directly. It’s clean, intuitive, and ready for real-world development. 


$ juno build hello.juno -j
$ juno exec hello
Hello, World!

$ java -cp "/bin/juno/lib" hello
