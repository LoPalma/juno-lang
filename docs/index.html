Meet Juno

Modern and simple JVM language

Installation

Get started with Juno by running the installer: 

bash <(curl -fsSL https://raw.githubusercontent.com/LoPalma/juno-lang/master/install.sh)

To install Juno on windows you must manually compile it from source, or you can use WSL and then run the installer.

Get Started

Check out how to write your first Juno program by visiting the docs.

Old look, new feel

Juno’s syntax will feel familiar to C programmers, but it’s designed to go far beyond. With support for variant types, pattern-like flexibility, and simpler semantics, Juno combines the elegance of modern languages with the performance of native code. 

import Io;

void main() {
  string|int x = 0; // integer
  x = "string"; // string
}

Slimware oriented

In Juno, modules replace the heavy class systems of other languages. You import functionality directly and operate on your data without boilerplate. It’s cleaner, lighter, and more modular—perfect for Slimware development. 

import String;
import Io;

void main() {
  string hi = "hi!";
  int len = String.length(hi);
}

Modern tools

Juno’s compiler is simple yet powerful. Run your program directly with juno run or export clean Jasmin bytecode. You can even execute Juno-generated classes with Java itself—no setup, no pain. 

# bash
$ juno build hello.juno -j
$ juno exec hello
Hello, World!

$ java -cp "/bin/juno/lib" hello
